# 快捷键速查

> 查看全部：`Ctrl+P` → 输入 `快捷键` → 打开快捷键面板。

## 编辑

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+N` | 新建笔记 |
| `Ctrl+Enter` | 在当前行下方另起一行 |
| `Ctrl+Shift+Enter` | 在当前行上方另起一行 |
| `Ctrl+D` | 删除当前行 |
| `Ctrl+[` / `Ctrl+]` | 缩进/取消缩进 |
| `Ctrl+Shift+K` | 切换任务状态 `[ ]` ↔ `[x]` |
| `Alt+↑/↓` | 上下移动当前行 |
| `Ctrl+Z` / `Ctrl+Shift+Z` | 撤销/重做 |

## 导航

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+O` | 快速切换笔记 |
| `Ctrl+P` | 命令面板 |
| `Ctrl+Home/End` | 跳到笔记开头/末尾 |
| `Ctrl+Alt+←/→` | 后退/前进（浏览历史） |

## 链接操作

| 快捷键 | 功能 |
|--------|------|
| 输入 `[[` | 插入/搜索内部链接 |
| `Ctrl+悬停` | 预览链接内容 |
| 点击链接 | 跳转到目标笔记 |
| 拖拽笔记到编辑区 | 快速插入链接 |

## 面板

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+E` | 切换编辑/阅读模式 |
| `Ctrl+Shift+F` | 搜索所有文件 |
| `Ctrl+G` | 打开关系图谱 |
| `Ctrl+Shift+L` | 切换左侧边栏 |
| `Ctrl+Shift+R` | 切换右侧边栏 |

## 搜索与替换

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+F` | 当前笔记内搜索 |
| `Ctrl+H` | 当前笔记内查找并替换 |
| `Ctrl+Shift+F` | 全局搜索 |

---

> 提示：在 `设置 → 快捷键` 中可自定义任意快捷键。
