# 模板与自动化

## 核心模板插件

Obsidian 内置 **模板 (Templates)** 插件，在 `设置 → 核心插件 → 模板` 中启用。

### 使用方法

1. 指定模板文件夹（如 `模板/`）
2. 在模板文件夹中创建 `.md` 模板文件
3. 新建笔记时 `Ctrl+P` → `插入模板`

### 模板变量

```
{{title}}          → 当前笔记标题
{{date}}           → 当前日期
{{time}}           → 当前时间
```

### 模板示例

```
---
tags: [项目]
created: {{date}}
---

# {{title}}

## 目标

## 任务

- [ ] 

## 备注
```

## Daily Notes（日记）

`设置 → 核心插件 → 日记` 开启。

- 指定日记文件夹（如 `日记/`）
- 点击左侧日历图标创建当天日记
- 定义日期格式（默认 `YYYY-MM-DD`）

### 日记模板

在模板文件夹创建日记模板，在日记设置中选择该模板：

```
---
tags: [日记]
---

# {{date}}

## 今日任务
- [ ] 

## 笔记
```

## Templater 插件（进阶）

社区插件 **Templater** 提供更强大的模板语法。

### 常用 Templater 语法

```
<% tp.date.now("YYYY-MM-DD") %>        → 当前日期
<% tp.file.title %>                     → 文件名
<% tp.file.cursor(1) %>                 → 插入后光标停留位置
<% tp.file.move("/目标/路径/" + ...) %>  → 自动移动到指定目录
<% tp.frontmatter.tags %>              → 读取当前文件的 tags 属性
```

### 常用模板：自动移动到收件箱

```
<% await tp.file.move("收件箱/" + tp.file.title) %>
```

创建笔记后自动移入收件箱。

## QuickAdd 插件

**QuickAdd** 提供快速捕获和宏命令：

- 一键创建笔记（含模板）
- 自定义捕获命令并绑定快捷键
- 执行 JS 脚本

**场景**：按 `Ctrl+Q` 弹出菜单 → 选"快速记录" → 内容自动追加到今日日记。
