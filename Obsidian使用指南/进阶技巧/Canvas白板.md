# Canvas 白板

Canvas 是 Obsidian 内置的无限画布，用可视化的方式组织笔记、图片和卡片。

## 创建方式

- `Ctrl+P` → `Canvas: 新建画布`
- 或在文件夹右键 → 新建画布

Canvas 文件以 `.canvas` 为扩展名（JSON 格式）。

## 添加内容

| 操作 | 方法 |
|------|------|
| 添加笔记 | 从左侧文件列表拖入 |
| 添加卡片 | 双击空白处，直接写 Markdown |
| 添加图片 | 拖入或粘贴 |
| 添加网页 | 粘贴 URL |
| 添加 Mermaid | 卡片中写 ` ```mermaid ` 代码块 |
| 嵌入笔记 | 拖入时自动 `![[笔记]]` |

## 连线

- 悬停卡片边缘出现圆点，拖拽到另一个卡片即可连线
- 点击连线可添加标签文字
- 右键连线可更改颜色和样式

## 分组

框选多个卡片 → 右键 → `创建分组`。分组可命名、着色、折叠。

## 适用场景

| 场景 | 用法 |
|------|------|
| 头脑风暴 | 卡片自由散落，快速记录想法，然后连线 |
| 项目规划 | 用卡片代表任务，分阶段分组 |
| 知识地图 | 笔记拖入画布，连线表示关系 |
| 写作大纲 | 卡片做章节，连线表示叙事顺序 |
| 读书笔记 | 卡片记要点，连线表示论证脉络 |

## 技巧

- 右键画布空白处 → `导出为图片` 可分享
- Canvas 内的卡片修改**不会**同步回原笔记，需手动更新
- 按住空格键拖拽可平移画布
