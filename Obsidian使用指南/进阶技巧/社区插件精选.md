# 社区插件精选

插件是 Obsidian 的灵魂。以下是按场景分类的精选推荐。

## 效率增强

| 插件 | 一句话说明 |
|------|-----------|
| **QuickAdd** | 一键捕获、模板插入、宏命令 |
| **Commander** | 自定义按钮、菜单、快捷键 |
| **Recent Files** | 最近打开文件列表 |

## 数据查询

| 插件 | 一句话说明 |
|------|-----------|
| **Dataview** | 用查询语言从笔记中提取数据，生成表格/列表/统计 |
| **DB Folder** | 类 Notion 数据库视图 |
| **Projects** | 项目管理面板 |

## 视觉增强

| 插件 | 一句话说明 |
|------|-----------|
| **Excalidraw** | 手绘风格白板，功能强大 |
| **Advanced Tables** | 表格编辑增强，自动对齐 |
| **Style Settings** | 主题参数可视化调节 |

## 内容管理

| 插件 | 一句话说明 |
|------|-----------|
| **Calendar** | 日历视图，联动 Daily Notes |
| **Kanban** | 看板视图，拖拽管理任务 |
| **Note Refactor** | 按标题拆分笔记 |
| **Tag Wrangler** | 批量重命名、合并标签 |

## 阅读与导出

| 插件 | 一句话说明 |
|------|-----------|
| **Better Word Count** | 选中文字数统计 |
| **Pandoc Plugin** | 导出 Word/PDF/LaTeX |
| **Copy as HTML** | 复制为富文本 |

## 安装方法

`设置 → 社区插件 → 浏览` → 搜索插件名 → 安装 → 启用。

> 建议：按需安装，不要一次性装太多。装完一个、用好一个、再装下一个。
