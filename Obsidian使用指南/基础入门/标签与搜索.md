# 标签与搜索

## 标签

### 基本用法

```
#标签名
#tutorial
#obsidian/基础     ← 嵌套标签（子标签）
```

- 在正文中任意位置输入 `#` + 文字即可
- 也可写在 Properties 的 `tags` 字段中

### 嵌套标签

```
#编程/Python
#编程/JavaScript
#项目/个人
#项目/工作
```

在标签面板中会层级展开，便于分类浏览。

### 标签 vs 链接

| | 标签 `#tag` | 链接 `[[ ]]` |
|---|---|---|
| 用途 | 分类、类型标记 | 关联具体笔记 |
| 粒度 | 粗（一类笔记） | 细（一对一关联） |
| 举例 | `#读书笔记` | `[[原子习惯]]` |

> 经验法则：**概念/实体用链接，类别/状态用标签**。

## 搜索

### 基础搜索

`Ctrl+Shift+F` 打开全局搜索，输入关键词即可。

### 搜索运算符

| 运算符 | 示例 | 说明 |
|--------|------|------|
| `path:` | `path:基础入门` | 限定目录 |
| `tag:` | `tag:#tutorial` | 按标签筛选 |
| `file:` | `file:语法` | 按文件名 |
| `line:` | `line:Python` | 按行内容 |
| `section:` | `section:安装` | 按标题段落 |
| `[]` | `[书单]` | 搜索任务状态 |
| `-` | `-tag:#todo` | 排除结果 |

### 组合搜索

```
tag:#python path:项目/     → 项目目录下带 #python 标签的
file:日记 tag:#重要         → 文件名含"日记"且有 #重要 标签
```

### 保存搜索

搜索框输入条件后点击"保存"，常用搜索变为书签。
