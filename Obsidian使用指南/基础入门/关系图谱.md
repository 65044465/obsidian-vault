# 关系图谱

图谱将笔记之间的 `[[链接]]` 可视化为节点和连线。

## 打开方式

- 左侧功能区 → **"关系图谱"** (Graph view)
- 或 `Ctrl+G`

## 全局图谱 vs 局部图谱

| | 全局图谱 | 局部图谱 |
|---|---|---|
| 打开 | 左侧面板点图谱图标 | 笔记右上角 `...` → 打开局部图谱 |
| 范围 | 整个仓库 | 仅当前笔记的关联 |
| 用途 | 鸟瞰知识结构 | 聚焦当前主题 |

## 交互操作

- **拖拽节点**: 重新布局
- **滚轮缩放**: 放大/缩小
- **悬停节点**: 显示笔记标题
- **点击节点**: 高亮其全部关联链接
- **拖拽空白区域**: 平移画布

## 实用设置（图谱右上角齿轮）

| 设置 | 建议 |
|------|------|
| 显示标签 | 开启，按标签分组颜色 |
| 显示附件 | 关闭，减少杂乱 |
| 孤立节点 | 开启，发现未链接的笔记 |
| 箭头 | 可选，看个人偏好 |

## 分组与筛选

在设置中可添加筛选条件：

```
- tag:#todo       → 只显示含 #todo 标签的笔记
- path:项目/       → 只显示"项目"目录下的笔记
```

可为不同分组设置不同颜色，快速识别笔记类型。
