# 双向链接

双向链接是 Obsidian 最核心的特性。一个笔记链接另一个，两者都会记录这个连接。

## 创建链接

```
[[笔记名]]              → 链接到笔记
[[笔记名|显示文本]]      → 自定义显示文字
[[笔记名#标题]]          → 链接到笔记内的某个标题
[[笔记名^段落ID]]        → 链接到笔记内的某个段落/块
```

输入 `[[` 时自动弹出建议列表，支持模糊搜索。

## 反向链接

打开任意笔记，右侧面板显示 **"反向链接"**：列出了所有链接到当前笔记的页面。

这是双向链接的另一半——不仅知道"我连了谁"，也知道"谁连了我"。

## 嵌入内容

在 `[[ ]]` 前加 `!` 即可嵌入目标内容：

```
![[笔记名]]              → 嵌入整个笔记
![[笔记名#标题]]          → 嵌入指定段落
![[图片.png]]             → 嵌入图片
![[音频.mp3]]             → 嵌入音频
```

## 链接未创建的笔记

在 `[[ ]]` 中写一个不存在的笔记名，它会显示为红色。点击即可自动创建该笔记——这是 Zettelkasten 方法的核心。

## 悬停预览

按住 `Ctrl` 并将鼠标悬停在链接上，可在弹窗中预览目标笔记内容，无需跳转。
