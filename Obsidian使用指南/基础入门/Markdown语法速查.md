# Markdown 语法速查

Obsidian 兼容标准 Markdown 并扩展了更多语法。

## 基础排版

```
# 一级标题
## 二级标题
### 三级标题

**粗体**
*斜体*
~~删除线~~
==高亮==
```

## 链接与图片

```
[[笔记名]]              → 内部链接
[[笔记名|显示文本]]      → 别名链接
[[笔记名#标题]]          → 链接到段落
[[笔记名^块ID]]          → 链接到块
[外部链接](https://...)  → 外部链接
![[图片.png]]            → 嵌入图片
![[笔记名]]              → 嵌入笔记内容
```

## 列表与任务

```
- 无序列表
1. 有序列表
- [ ] 待办事项
- [x] 已完成
```

## 表格

```
| 列1 | 列2 | 列3 |
|-----|-----|-----|
| A   | B   | C   |
```

## 代码

````
`行内代码`

```语言名
代码块（支持语法高亮）
```
````

## 引用与分隔

```
> 引用文字
> 多行引用

---
```

## 脚注

```
这是文字[^1]

[^1]: 这是脚注内容
```

## 数学公式 (LaTeX)

```
行内公式: $E = mc^2$

块级公式:
$$
\sum_{i=1}^{n} x_i
$$
```

## Callout 提示框

```
> [!note] 标题
> 普通提示内容

> [!warning] 警告
> 警告内容

> [!tip] 技巧
> 技巧内容

> [!info] / [!danger] / [!example] / [!quote]
```
