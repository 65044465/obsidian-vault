# 笔记创建与管理

## 新建笔记

| 方式 | 操作 |
|------|------|
| Ctrl+N | 创建空白笔记 |
| 点击 [[ ]] 中的红链接 | 自动创建并打开目标笔记 |
| 右键文件夹 → 新笔记 | 在指定位置创建 |

## 快速切换

`Ctrl+O` — 输入关键词，快速跳转到任意笔记。

## 文件操作

- **重命名**: 右键笔记 → 重命名（链接会自动更新）
- **移动**: 拖拽到目标文件夹（链接会自动更新）
- **删除**: 右键 → 删除（会进入 `.trash` 回收站）

> Obsidian 会自动更新所有 `[[链接]]`，无需手动修复。

## 属性 (Properties / Frontmatter)

文件顶部的 YAML 元数据，支持自定义字段：

```
---
tags: [入门, 教程]
created: 2026-05-23
aliases: [笔记管理]
---
```

- `tags`: 标签
- `aliases`: 别名（搜索时可用别名匹配）
- 可添加自定义属性，配合 Dataview 等插件使用

## 文件结构建议

```
Vault/
├── 00-目录.md          ← MOC 索引笔记
├── 收件箱/             ← 快速记录，定期整理
├── 基础入门/           ← 按主题分类
├── 进阶技巧/
├── 日常/               ← Daily Notes
└── 附件/               ← 图片、PDF 等
```

### 命名建议

- 使用有意义的标题而非 `Untitled 1`
- 可加日期前缀：`2026-05-23 会议记录`
- 保持一致即可，无需过度设计
